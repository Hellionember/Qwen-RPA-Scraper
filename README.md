# Qwen RPA Scraper

Автоматизированный робот (RPA) на базе **Python** и **Selenium** для извлечения характеристик товаров из веб-интерфейса LLM **Qwen** (`chat.qwen.ai`).

---

## В чем суть проекта?

Платные API крупных языковых моделей (OpenAI, Anthropic, Alibaba Cloud) тарифицируются по количеству отправленных и полученных токенов. При обработке баз данных на десятки тысяч строк расходы на API могут составлять сотни и тысячи долларов.

**Решение:**
* Робот автоматизирует действия человека в браузере и использует **бесплатную веб-версию Qwen Chat**.
* Он самостоятельно обходит защиту от зависаний, интерактивные опросы интерфейса («выбор лучшего ответа») и системные сбои веб-интерфейса.
* Проект позволяет обрабатывать терабайты номенклатур абсолютно **бесплатно**, используя умные алгоритмы группировки и батчинга.

---

## Подготовка и установка

### 1. Окружение
Рекомендуется использовать изолированное окружение **Conda** (Python 3.10 или 3.11):
Создайте окружение, активируйте его и установите все необходимые библиотеки одной командой с помощью файла `requirements.txt` (он уже находится в репозитории):

```bash
conda create -n qwen_rpa python=3.10
conda activate qwen_rpa
pip install -r requirements.txt
```

### 2. Установка веб-драйвера Edge
Для работы робота требуется браузер Microsoft Edge и подходящий для него драйвер:
1. Проверьте версию вашего Edge (Меню -> Справка и отзывы -> О программе).
2. Скачайте соответствующий **EdgeDriver** с официального сайта [Microsoft Edge Developer](https://developer.microsoft.com/en-us/microsoft-edge/tools/driver/).
3. Распакуйте архив и укажите путь к файлу `msedgedriver.exe` в файле конфигурации.

## Настройка проекта перед запуском

В корневой папке проекта необходимо создать два конфигурационных файла:

### 1. Создание файла `.env` (Секретные данные)
Создайте файл `.env` в корневом каталоге и впишите свои учетные данные от аккаунта Qwen:
```env
QWEN_EMAIL=your_email@example.com
QWEN_PASSWORD=your_secure_password
```

### 2. Создание файла `config.txt`
Скопируйте файл `config.example.txt` под именем `config.txt` и укажите ваш путь к EdgeDriver:
```text
EDGE_DRIVER_PATH = C:\path\to\your\msedgedriver.exe
INPUT_EXCEL = Входной.xlsx
OUTPUT_EXCEL = Выходной.xlsx
```

## Структура входного файла Excel (`Входной.xlsx`)

Поскольку исходные файлы данных содержат коммерческую тайну, они не публикуются в репозитории. Для успешного запуска робота ваш входной файл `Входной.xlsx` должен содержать строго следующие 4 листа:

### Лист 1: `Материалы`
Очередь задач для обработки. Колонки:
* `ид_исторический` - уникальный ID записи.
* `наименование` - текстовое описание товара (например, *«Лоток перфорированный SPB-RF 100»*).
* `класс` - числовой код категории товара.
* `бренд` - название производителя (должно совпадать со справочником).

### Лист 2: `Класс-Признаки`
Спецификация характеристик для каждого класса. Колонки:
* `класс` - числовой код категории (соответствует листу Материалы).
* `признак` - название характеристики, которую Qwen должен извлечь (например, *«Высота, мм»*, *«Покрытие»*, *«Толщина, мм»*).

### Лист 3: `Справочник брендов`
База знаний по брендам. Колонки:
* `Бренд` - название бренда.
* `Сайт` - официальный сайт производителя. Qwen использует его для контекстного поиска информации, что исключает галлюцинации.

### Лист 4: `Шаблон промта`
В ячейке **A1** должен быть записан текстовый шаблон запроса к модели с использованием переменных тегов:
```text
Требуется заполнить значения признаков в табличной форме.
Дай ответ в csv с разделителем «€», названия столбцов таблицы с результатами:
Номер, наименование, <А1 - взять по классу Класс-признаки>, твоя уверенность в результате (%)
Используй только данные с сайта <А3 - взять по бренду Справочник брендов>
Номер Наименование
<А2 - взять 500 материалов или меньше по классу + бренду>
```

## Запуск

После настройки файлов `.env`, `config.txt` и подготовки файла `Входной.xlsx` запустите скрипт:
```bash
python Qwen_brand_5.py
```

Робот автоматически:
1. Запустит скрытый или видимый браузер Edge.
2. Пройдет авторизацию в Qwen Studio.
3. Проверит уже обработанные ранее ID в файле результатов (поддерживается дозапись при прерывании).
4. Начнет отправку пакетов данных, извлекая характеристики по строгому позиционному Regex-алгоритму.
5. Запишет результаты в базу SQLite и экспортирует их в итоговый Excel.
